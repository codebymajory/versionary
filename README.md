
# Taxi Identification (Expo PWA)

Frontend-only prototype for a progressive web app built with Expo (React Native Web).
Implements:
- Install prompt (Android) or iOS "Add to Home Screen" instructions
- Onboarding -> KYC form -> Verifying -> Dashboard (simulated)
- Responsive layout and Montserrat font
- Cameroon phone validation for MTN/Orange prefixes

## Quick start

```bash
npm i -g expo@latest
npm install
npm run web
```

When ready to ship a static PWA build:
```bash
npm run build:web
# Output will be in dist/ (Expo 51+)
```
